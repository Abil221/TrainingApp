- `workout_plan_card.dart` - красивые карточки планов

**Экран:**

## 🔍 ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ

### Разблокировка достижения:
```dart
await achievementService.unlockAchievement(userId, achievementId);
```

### Добавление XP:
```dart
await achievementService.addXp(userId, 50);  // +50 XP
```

### Создание плана:
```dart
await workoutPlanService.createPlan(
  userId: userId,
  name: 'Недельный сплит',
  description: 'Грудь, спина, ноги',
  durationWeeks: 4,
);
```

### Создание цели:
```dart
await goalService.createGoal(
  userId: userId,
  goalType: GoalType.weightLoss,
  name: 'Похудеть на 10 кг',
  targetValue: 80,
  currentValue: 90,
  unit: 'кг',
  deadline: DateTime.now().add(Duration(days: 90)),
);
```

### Запись веса:
```dart
await goalService.recordWeight(userId, 85, notes: 'После тренировки');
```
